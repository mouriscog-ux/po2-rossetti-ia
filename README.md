# 🏆 Disco Brasileirão 2026

Simulador de Air Hockey arcade com temática do Campeonato Brasileiro Série A de 2026.

## 📁 Estrutura do Projeto
- `index.html`: Ponto de entrada e interface do usuário.
- `css/style.css`: Estilização premium e efeitos visuais.
- `js/`: Lógica do jogo (Física, IA, Core).

## 🎮 Mecânicas
- **Torneio Mata-Mata**: Oitavas, Quartas, Semi e Final.
- **Dificuldade Adaptativa**: Fácil, Médio e Difícil.
- **Física Realista**: Inércia, fricção e colisão elástica.
